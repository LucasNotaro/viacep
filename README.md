# Consulta de CEP

App React Native para consultar endereços via CEP usando a API do ViaCEP.

## Como usar

1. Digite um CEP válido
2. Pressione "Buscar"
3. Veja as informações do endereço

## Tecnologias

- React Native + Expo
- React Native Paper
- ViaCEP API

## Executar

```bash
npm install
npm start
```

```
viacep/
```
